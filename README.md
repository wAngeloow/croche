<div align="center">
  <h1>E-Commerce de Crochê</h1>
  <p>Este projeto é um site de e-commerce de crochê, onde apresenta diversos tipos de crochês baseado em categorias.
  </p>
</div>

## Funcionalidades
- Design Responsivo: O site se adapta a todos os dispositivos (desktop, tablet e mobile).  
- Navegação Dinâmica: O conteúdo muda conforme a categoria escolhida, sem precisar recarregar a página.  
- Catálogo Interativo: Exibição de produtos com imagem, descrição, preço e botão de compra.  
- FAQ (Perguntas Frequentes): Uma seção dedicada para esclarecer dúvidas do usuário.  
- Contato Direto: Links para redes sociais e um botão fixo de WhatsApp para facilitar o atendimento.  
- Página Inicial Personalizada: Mostra imagens das categorias antes de exibir os produtos.  

## Tecnologias
- HTML  
- CSS
- Bootstrap
- JavaScript
